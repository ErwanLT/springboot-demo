# Authorization Server OAuth2

Serveur d'autorisation OAuth2 basé sur Spring Authorization Server.

## Description

- Fournit l'issuer OAuth2/ OIDC.
- Sert de source de jetons pour les clients et le resource server.

## Configuration clé

- Port : `9000`
- Issuer : `http://localhost:9000`

## Comment l'exécuter

```bash
mvn spring-boot:run
```
