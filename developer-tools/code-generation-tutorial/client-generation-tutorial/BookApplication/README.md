# Book Application

Application Spring Boot qui expose une API de gestion des livres.

## Endpoints

- `GET /books`
- `GET /books/{id}`
- `GET /books/author/{id}`
- `POST /books`
- `PUT /books/{id}`
- `DELETE /books/{id}`

## Configuration clé

- Port : `8092`

## Comment l'exécuter

```bash
mvn spring-boot:run
```
